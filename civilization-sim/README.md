# 🏛️ Civ Sim — 文明模拟器

纯前端文明模拟器，打开浏览器即可运行。

## 功能

- **程序化世界地图**：Perlin噪声地形生成（海洋、平原、森林、山地、沙漠、雪地）
- **领土扩张**：部落从1格起步，随人口自然扩张
- **经济系统**：食物/木材/石材/铁矿 + 分工（猎人/农民/矿工/学者）
- **建筑系统**：农场、矿场、兵营、学院、市场、城墙
- **科技树**：15级（石器→太空），研究加速2倍
- **战争系统**：军事力=人口×武器科技，领土割让
- **人口动态**：饥荒衰退、腐败分裂
- **时代视觉**：地图色调/城市图标/道路网络随时代变化

## 运行

直接用浏览器打开 `index.html` 即可。

## 技术实现

- `index.html` — 页面+CSS
- `engine.js` — 模拟引擎（1000+行）
- `renderer.js` — Canvas渲染器
