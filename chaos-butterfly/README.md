# 🦋 Chaos Butterfly — 混沌理论可视化

用交互式可视化探索混沌系统，拖动参数看蝴蝶效应。

## 功能

- **洛伦兹吸引子**：经典三体混沌系统
- **逻辑映射**：Logistic Map 分岔图
- **元细胞自动机**：Conway's Game of Life + 预设规则
- **参数控制**：拖动滑块实时改变系统行为

## 运行

纯前端，打开 `index.html` 即可。每个可视化也可独立打开。

## 文件

- `index.html` — 洛伦兹吸引子（主页面）
- `logistic-map.html` + `logistic-map.js` — 逻辑映射分岔图
- `cellular-automata.html` — 元细胞自动机
- `life-game-engine.html` — 生命游戏引擎
