# 🏠 Lab Homepage — 实验室首页

开源实验室的主页，展示所有项目入口。

## 设计风格

- 米白底 + 纸质纹理（整体）
- 每张项目卡片对应各自项目的视觉风格：
  - Civ Sim：羊皮纸质感和角落装饰
  - Evo Music：深紫 + 声波线条
  - Video Prompt：暖琥珀 + 胶片条装饰
  - Quantum Crypto：深蓝 + 量子网格
  - Palette Shaker：暖白 + 调色盘圆点 + 底部彩虹条
  - Chaos Butterfly：深色 + 双色漩涡 + 无穷符号
  - Digital Garden：深绿 + 萤火虫动画

## 运行

纯前端，打开 `index.html` 即可。
