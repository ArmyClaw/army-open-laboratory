# 🌿 Digital Garden — 交互式电子花园

程序化生成300+种花卉的SVG花园，晨昏主题切换，花语图鉴。

## 功能

- **程序化SVG花卉**：20种渲染风格（玫瑰/雏菊/荷花/向日葵等）
- **300+种花数据**：每种有中英文名、科属、颜色、花语、趣事
- **默认16朵**：页面加载时显示16朵，可点击"种下更多花"
- **花选择器**：+10/+50/+100/全部，新花带绽放动画
- **晨昏主题**：黎明/白天/午后/黄昏/夜晚五种主题
- **花语图鉴**：点击花朵查看详细信息
- **动态装饰**：萤火虫、蝴蝶、蘑菇、苔藓、鹅卵石

## 运行

纯前端，打开 `index.html` 即可。花数据从 `garden.json` 加载。

## 技术实现

- 花朵通过JS Canvas/SVG程序化生成，无需图片素材
- 使用seeded random确保确定性渲染
- CSS变量驱动主题切换
