## 项目列表

| # | 项目 | 目录 | 简介 |
|---|------|------|------|
| 01 | 🏛️ Civ Sim | [civilization-sim/](civilization-sim/) | 纯前端文明模拟器。程序化地形、经济、人口、科技树、领土扩张、战争。 |
| 02 | 🧬 Evo Music | [evo-music/](evo-music/) | 进化音乐。马尔可夫链生成旋律 + 遗传算法进化，AABA曲式编曲。 |
| 03 | 🎬 Video Prompt | [video-prompt-gen/](video-prompt-gen/) | 专业级AI视频提示词生成器。相机运动、电影风格、镜头语言。 |
| 04 | 🔐 Quantum Crypto | [post-quantum-crypto/](post-quantum-crypto/) | 后量子密码学实验室。RSA加密、Shor算法、格密码、安全检测器。 |
| 05 | 🎨 Palette Shaker | [palette-shaker/](palette-shaker/) | PPT配色搭配器。输入品牌色生成整套配色，实时预览。 |
| 06 | 🦋 Chaos Butterfly | [chaos-butterfly/](chaos-butterfly/) | 混沌理论可视化。洛伦兹吸引子、逻辑映射、元细胞自动机、生命游戏。 |
| 07 | 🌿 Digital Garden | [digital-garden/](digital-garden/) | 交互式电子花园。300+种程序化SVG花卉、晨昏主题切换、花语图鉴。 |

## 技术栈

- **纯前端**：所有项目均为纯HTML/CSS/JS，无需后端服务器
- **零依赖**：不使用任何外部框架或库（除Google Fonts）
- **Canvas渲染**：模拟器和可视化均基于Canvas 2D
- **程序化生成**：地形、花卉、粒子系统全部算法生成
