# 🎬 Video Prompt Gen — AI视频提示词生成器

专业级AI视频提示词生成工具，覆盖相机运动、电影风格、镜头语言。

## 功能

- **相机运动**：推拉摇移跟、航拍、手持等20+种运动
- **电影风格**：胶片颗粒、色调、宽高比
- **镜头语言**：特写/中景/远景、构图规则
- **平台导出**：适配Sora/Runway/Pika/Kling等平台格式
- **风格预设**：赛博朋克、自然纪录片、复古胶片等

## 运行

纯前端，打开 `index.html` 即可。
