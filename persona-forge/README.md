# Persona Forge — 角色人设生成器

一个交互式工具，选择影视角色、历史名人等，自动生成完整的 AGENTS.md 格式人设文档。

## 功能

- **30+ 内置角色**，按类别分组：影视、历史、流行文化、虚构角色
- **人设生成**：语言特征、行为准则、互动规则、标志性口号、关联人物、比喻素材、禁止准则、使用建议
- **打字机效果**：逐字渲染生成文档
- **混合角色**：选两个角色生成融合人设（彩蛋功能）
- **复制/下载**：一键复制 Markdown 或下载 .md 文件
- **名言轮播**：底部持续滚动角色名言

## 设计风格

电影片场/编剧工作台风格：深色木纹背景 + 暖光聚光灯 + 牛皮纸效果。

## 文件结构

```
├── index.html          # 页面 + CSS
├── persona-data.js     # 角色数据库 (30+ 角色)
├── generator.js        # 人设文档生成引擎
```
